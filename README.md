# RequestPro (Modelio Module)

Bu repo **sadece yayınlanmış `.jmdac` modül dosyalarını** dağıtmak için kullanılır.  
Kaynak kod bu repoda bulunmaz.

## Gereksinimler

- **Modelio 5.4** (modülü çalıştırmak için)

> Not: Modülün uyumluluğu Modelio sürümüne bağlıdır. Lütfen **Modelio 5.4** kullanın.

## İndirme

En güncel sürümü ve önceki sürümleri buradan indirebilirsiniz:

- GitHub → **Releases** sayfası  
  Her release altında `.jmdac` dosyası **asset** olarak eklenir.

## Kurulum

1. GitHub **Releases** bölümünden `.jmdac` dosyasını indirin.
2. Modelio 5.4’ü açın.
3. Modelio içinden modül kurulum/ekleme ekranına gidip indirdiğiniz `.jmdac` dosyasını seçerek yükleyin.
